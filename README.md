# VeloGuardian

**Fast, private VPN protection for businesses and individuals.**

VeloGuardian is a cybersecurity company building network security products powered by WireGuard. Our core product is a no-log VPN with built-in threat filtering, supported by self-hosted network security appliances.

## Products

### VeloGuardian VPN

WireGuard-powered VPN with Cloud Shield threat filtering built in. Native apps for Windows, macOS, iOS, and Android.

- WireGuard encryption with zero-log policy
- Cloud Shield: malware, phishing, and botnet filtering at the network level
- Per-user management dashboard
- Unlimited devices per subscription

**[Learn more](https://veloguardian.com/features.html)** | **[Download](https://veloguardian.com/download.html)** | **[Pricing](https://veloguardian.com/pricing.html)**

---

### VeloGuardian DNS

**Free, open-community** DNS filtering appliance for your network.

- 90+ filtering categories
- Block ads, malware, phishing, and trackers network-wide
- Self-hosted virtual appliance (OVA) — your data stays on your infrastructure
- Web dashboard for configuration and monitoring
- No account required, no telemetry, completely free

**[Learn more](https://veloguardian.com/dns.html)** | **[Quick Start Guide](https://veloguardian.com/docs/dns/quick-start.html)**

> VeloGuardian DNS is our free community product. We welcome bug reports, feature requests, and feedback via [GitHub Issues](https://github.com/veloguardian/community/issues) and [Discussions](https://github.com/veloguardian/community/discussions).

---

### VeloGuardian NetGuard

Managed WireGuard gateway appliance for secure remote access to your office LAN.

- Web-managed WireGuard VPN gateway
- Deploy as a virtual appliance on your network
- Secure remote access for your team from anywhere
- Integrated with VeloGuardian VPN subscriptions

**[Learn more](https://veloguardian.com/netguard.html)**
